Assignment

Goal

As a user, I want to see my public photos

As a dev, query the Flickr API to only display photos from my user ID. Hint your user ID is located in the API explorer. Meaning when you find the correct Flickr API method, go to the API explorer at the bottom, and your user ID will be on the right panel.

Expectation

Both the server (node.js) and the client (browser) should show the same photos. Upload at least two photos to your Flickr photostream.

Technologies

  • Flickr API
  • JavaScript
  • jQuery
  • jQuery AJAX
  • node.js
  • hapi.js
